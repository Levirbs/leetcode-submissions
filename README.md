# LeetCode Solutions

Repositório com minhas resoluções de problemas de algoritmos e estruturas de dados no NeetCode.

## Sobre

Este projeto reúne meu histórico de prática em problemas clássicos de entrevistas técnicas,
com foco em raciocínio algorítmico, escolha de estruturas de dados e evolução de solução.

Em vez de manter apenas uma resposta final por problema, o repositório preserva diferentes tentativas,
o que ajuda a visualizar ganhos de clareza, desempenho e organização do código ao longo do tempo.

## Fluxo automático

As submissões são sincronizadas automaticamente pelo GitHub Sync do NeetCode.

Na prática, isso elimina manutenção manual: quando uma nova submissão é feita, o histórico pode ser atualizado
automaticamente no repositório, mantendo consistência sem exigir ajustes frequentes no conteúdo do projeto.

Esse fluxo permite concentrar energia no que realmente importa: resolver problemas, revisar abordagens e evoluir
de forma contínua.


## Estrutura

As soluções são organizadas por tema e por problema. Cada tentativa fica salva como `submission-x`,
preservando o contexto de evolução de cada questão.

Com essa organização, fica fácil navegar por tópicos, comparar tentativas de um mesmo problema e acompanhar
mudanças de estratégia entre versões.

```text
Data Structures & Algorithms/
  <problema>/
    submission-0.ext
    submission-1.ext
    ...
```

## Tecnologias

- **Linguagem**: C++ 21
- **Plataforma**: NeetCode

As implementações priorizam uso moderno da linguagem, legibilidade e eficiência, refletindo prática orientada
a entrevistas e fundamentos sólidos de algoritmos.
